# Vertex Estratégia Digital

Site institucional da Vertex Estratégia Digital, uma empresa especializada em estratégias digitais completas para transformar negócios.

## Tecnologias Utilizadas

- React 18
- Vite
- Tailwind CSS
- Framer Motion
- Inter Font (Google Fonts)

## Como Executar

1. Instale as dependências:
   ```bash
   npm install
   ```

2. Execute o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

3. Abra [http://localhost:5173](http://localhost:5173) no navegador.

## Estrutura do Projeto

- `src/components/` - Componentes React
- `src/` - Arquivos principais (App.jsx, main.jsx, index.css)
- `public/` - Assets estáticos (logo_vertex.jfif, Vertex_banner.png)

## Deploy

Para build de produção:
```bash
npm run build
```

Os arquivos serão gerados na pasta `dist/`.
